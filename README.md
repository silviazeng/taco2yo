## 🚀 Host it free on GitHub Pages

1. Create a new repo on GitHub (e.g. `taco-birthday`).
2. Upload `index.html` (and this README) to the repo — either drag-and-drop on github.com, or:
   ```bash
   git init
   git add .
   git commit -m "Taco's 2nd birthday site"
   git branch -M main
   git remote add origin https://github.com/YOUR-USERNAME/taco-birthday.git
   git push -u origin main
   ```
3. On GitHub: **Settings → Pages**.
4. Under **Build and deployment → Source**, choose **Deploy from a branch**.
5. Branch: **main**, folder: **/ (root)** → **Save**.
6. Wait ~1 minute. Your site will be live at:
   `https://YOUR-USERNAME.github.io/taco-birthday/`

Share that link with guests! 🎉
